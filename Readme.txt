Snake N Ladder
Note:-  we use system variables for sensitive information like mongoDB url, database user and passwordand other, but here I'm not using the system variables so that the team can run the application on their system.
----------------
1) Requirments
------------------
	a) NodeJS installed
	b) Internet connection

2) To start the app 
-------------------
	a) open CMD/terminal at the base directory and run "npm install"
	b) run command "npm start".
	c) "Connection Established At PORT: 3000" when terminal shows this message open browser and visit "http://127.0.0.1:3000".

3)npm libraries 
------------------
a) express
b) body-parser
c) random-number
d) connect-flash
e) ejs
f) express-session
g) underscore
